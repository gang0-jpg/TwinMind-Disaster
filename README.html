TwinMind-Disaster

TwinMind-Disaster is an experimental AI-driven Digital Twin platform for flood risk analysis and sensor optimization.

The system combines terrain analysis, flood simulation, deep learning (U-Net), and sensor placement optimization to support disaster prevention and urban resilience.

Concept

Modern cities require real-time disaster prediction and decision support.

TwinMind-Disaster explores a Digital Twin architecture where virtual simulations and AI models support real-world disaster management.

Main objectives:

  • Flood risk prediction
  • Optimal sensor placement
  • Digital Twin monitoring interface
  • AI-assisted disaster response

Features

Terrain Processing

  • DEM processing using GSI elevation data
  • slope map generation
  • terrain preprocessing for flood simulation

Flood Simulation

A lightweight flood solver generates synthetic flood scenarios.

Features:

  • rainfall scenario generator
  • terrain-based water flow simulation
  • dataset generation for AI training

Output format:

NPZ dataset
├ rain
├ dem
└ flood

Deep Learning Flood Prediction

A U-Net based CNN model predicts flood distribution from rainfall and terrain.

Input channels:

  • rainfall map
  • DEM elevation
  • slope map

Output:

  • flood probability / water depth

Framework:

  • PyTorch

Sensor Placement Optimization

Sensor positions are optimized using predicted flood maps.

Goal:

  • maximize information gain
  • minimize sensor count
  • improve monitoring coverage

Outputs:

sensor locations
coverage visualization
optimization logs

Digital Twin Dashboard

An interactive dashboard built with Streamlit.

Features:

  • terrain visualization
  • rainfall scenario display
  • flood prediction results
  • sensor optimization visualization

Run:

streamlit run ui/twinmind_dashboard.py

Installation

pip install -r requirements.txt

Data Preparation

Generate flood simulation cases:

python scripts/generate_cases.py

Output:

data/cases/*.npz

Model Training

Train the U-Net model:

python scripts/train_unet.py

Output:

runs/unet_best.pt

Prediction

Run flood prediction:

python scripts/predict_unet.py

Sensor Optimization

python scripts/optimize_sensors.py

Results:

runs/sensor_optimization/
