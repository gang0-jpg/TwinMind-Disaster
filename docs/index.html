Overview

The system predicts flood depth from terrain data and rainfall, estimates model uncertainty, and proposes effective sensor placement based on uncertainty and road-constrained candidate locations.

Core Technologies

  • U-Net for spatial flood-depth prediction
  • Reservoir computing for structural support
  • MC Dropout for uncertainty estimation
  • Sensor placement optimization using variance maps
Built for TwinMind-Disaster Project

Map data: Geospatial Information Authority of Japan (GSI)
Compute infrastructure: IO.net
